# 仿知乎日报
此app作为新闻客户端类应用的练习
以知乎日报和网易新闻客户端作为参考进行仿照
    
内容调用了知乎日报的API
[知乎日报api](https://github.com/izzyleung/ZhihuDailyPurify/wiki/知乎日报-API-分析)
    
    
##完成功能
启动动画页面
下拉刷新
最新文章列表和加载更多文章
轮播控件显示部分最新文章
侧滑菜单
顶部tab页加载其他主题内容
底部按键更换当前主页内容（内容部分将会后续添加）
文章内容展示（使用WebView）
文章内容的离线缓存
    
后续陆续添加
内容列表的缓存
收藏功能
